# THEATRON - Rehearsals

This directory contains rehearsal documentation, creative process records, and developmental materials.

## Structure

Rehearsal entries are organized chronologically with supporting metadata:

- Session notes and focus areas
- Objectives and outcomes
- Technical notes
- Development progress
- Next steps and planning

## Entry Format

Each rehearsal session should include:

- Date (MMDDYY format)
- Duration
- Focus area/topic
- Objectives
- Notes and observations
- Next steps for future sessions

## Purpose

This section documents the creative process, allowing visitors to see the development and evolution of performances over time.
