# THEATRON - Performances

This directory contains performance recordings, complete stage documentation, and theatrical presentation materials.

## Structure

Performance entries are organized chronologically with supporting metadata:

- Video files and links
- Performance notes and technical documentation
- Dates and duration information
- Venue information
- Audience information

## Entry Format

Each performance should include:

- Title
- Date (MMDDYY format)
- Duration
- Venue
- Description/Notes
- Technical notes

## Future Integration

This directory is prepared for integration with the **EnhancedVideoPlayer** component for streaming and embedded performance documentation.
